# Heap Data Structures: Implementation, Analysis, and Applications

## Overview

This project implements HeapSort and a Priority Queue using Python. The project demonstrates heap operations, analyzes algorithm complexity, and simulates task scheduling using a priority queue.

## Files

* heap_sort.py – HeapSort implementation
* priority_queue.py – Priority Queue implementation using a heap
* scheduler.py – Task scheduling simulation
* README.md – Project documentation

## Running the Programs

### HeapSort

```bash
python heap_sort.py
```

### Priority Queue

```bash
python priority_queue.py
```

### Scheduler Simulation

```bash
python scheduler.py
```

## Results

HeapSort successfully sorts arrays in ascending order.

Priority Queue successfully processes tasks according to priority.

Task Scheduler executes tasks from highest priority to lowest priority.

## Complexity Analysis

| Operation   | Complexity |
| ----------- | ---------- |
| Insert      | O(log n)   |
| Extract Min | O(log n)   |
| HeapSort    | O(n log n) |
| Build Heap  | O(n)       |
